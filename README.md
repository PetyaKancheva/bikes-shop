# bikes-shop
Web-app for a bicycle shop 🚴‍♂️

- ✅ Spring Boot project
- ✅ Role based access with Spring Security
- ✅ Events, Scheduelers, Cashe
- ✅ REST APIs
- ✅ Cookies

✏️ Planned deployment to Azure
✏️ Planned internationalisation
